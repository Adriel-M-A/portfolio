## Sobre el proyecto

Sistema de punto de venta desarrollado para pequeños y medianos comercios. Permite gestionar el flujo completo de una venta: desde la carga de productos hasta el cierre de caja diario.

## Características principales

- Registro y gestión de productos con stock
- Procesamiento de ventas con múltiples métodos de pago
- Generación de reportes diarios y mensuales
- Panel de administración para gestión de usuarios
- Historial de transacciones con filtros por fecha

## Desafíos técnicos

El mayor desafío fue el manejo del estado global de la caja en tiempo real, sincronizando las operaciones de venta con el inventario sin pérdida de datos ante cortes de conexión.
